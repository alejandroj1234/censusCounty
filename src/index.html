About Interactive Counties Map

Interactive Counties Map is a JavaScript based application which presents United States Census data in an interactive map and dynamic graphs.

Interactive Counties Map is configured in the back-end using Node.js for the platform and Express for the server. In the front-end Bootstrap is used for the user interface components and D3.js is used to render the Census Bureau data in various formats. The data is attained by calling the US Census Bureau JSON-based APIs.

Users can select from various demographic distribution criteria in order to see which counties match the selected criteria. Users are then able to continue selecting criteria, up to five in total, presenting only the counties which match all the criteria selected. Users can view all of the demographic criteria for a specific county in the form of various graphs by clicking on a county on the map.
